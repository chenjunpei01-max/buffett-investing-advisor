# Buffett Investing Advisor

下载 ZIP 并解压后，把解压出来的文件夹路径发给 Codex，让 Codex 帮你安装成 skill。

示例：

```text
请把这个文件夹安装成 Codex skill：
C:\Users\你的用户名\Downloads\buffett-investing-advisor-main
```

安装后新开 Codex 对话，输入：

```text
用 $buffett-investing-advisor 分析贵州茅台现在能不能买
```

## 参考数据来源

这个 skill 的投资原则优先参考 Berkshire Hathaway 官方材料：

- Berkshire Hathaway shareholder letters: https://www.berkshirehathaway.com/letters/letters.html
- Berkshire Hathaway Owner's Manual: https://www.berkshirehathaway.com/owners.html
- Owner-related business principles PDF: https://www.berkshirehathaway.com/ownman.pdf
- Berkshire Hathaway annual reports: https://www.berkshirehathaway.com/reports.html

分析具体股票时，Codex 会按需核验最新行情、年报、季报、公告、估值和监管信息；如果没有最新数据，输出应标注为初步倾向。
